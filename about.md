## What is UMRS? 

The Unclassified MLS Reference System (UMRS) is a free, MIT-licensed reference system for anyone who needs to
protect sensitive unclassified information — whether that means following CMMC, handling U.S. CUI designations,
working under Canadian Protected A/B/C requirements, or simply building software that a security auditor can actually
verify. UMRS covers the full stack: deploying kernel-enforced information labels, running security tools that give
operators evidence-backed answers, and writing high-assurance Rust with patterns that hold up under scrutiny. The
engineering is real. The controls are mapped. The source is open.

UMRS Project documentation will be avaiable on this site soon, so check back.

## Primary Project: UMRS

My primary project is the UNCLASSIFIED MLS Reference System Project.

UMRS is a hands-on, high-assurance engineering project built on Red Hat
Enterprise Linux 10 in FIPS mode. It demonstrates how a real organization can
design, implement, and operate a hardened Linux system using SELinux MLS and a
complete unclassified label hierarchy for Controlled Unclassified Information
(CUI), aligned with frameworks such as NIST SP 800-53 and the Cybersecurity
Maturity Model Certification (CMMC).

Within this project, we have a couple of projects exploring, and producing some
for interesting stuf.
